# Java Polymorphism

## Definition
- Same interface, different behavior

---

## Types

### Compile-time (Overloading)
Same method name, different parameters

### Runtime (Overriding)
Method resolved at runtime

---

## Example

Animal a = new Dog();
a.speak();

---

## Key Idea
- Runtime decides which method to call
- Enables flexible design

---

## Important Term
Dynamic Dispatch
